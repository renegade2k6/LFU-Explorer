Landscape System

System Overview

The Landscape System allows players to decorate their manor with 297 unique landscape decorations across 27 distinct categories. Each decoration can be upgraded through 11 levels (0-10), with visual evolution occurring at key progression points.

Core Statistics

  • Total Landscape Decorations: 297
  • Landscape Categories: 27
  • Levels Per Item: 11 (0-10)
  • Visual Evolution Tiers: 3 tiers

System Features

  • Visual Upgrades: Models evolve at Level 5 (tier 2) and Level 10 (tier 3)
  • Manor Building Requirement: "Please build the Manor first"
  • Upgrade System: Each decoration progresses through 11 levels with increasing stats
  • Bonus Effects: Decorations provide combat stat bonuses that take effect immediately when placed
  • Bonus Caps: Each decoration type has a maximum bonus limit; higher-level decorations take priority

Progression System

Level Structure

Each landscape decoration follows a standardized 11-level progression (0-10):

  • Early Game (Levels 0-4): Base visual model (_01 variant)
  • Mid Game (Levels 5-9): Enhanced visual model (_02 variant)
  • End Game (Level 10): Maximum visual model (_03 variant)

Visual Evolution Points

Level Milestone Model Change Impact
Level 0-4 Base Model (_01) Initial decoration appearance
Level 5 Enhanced Model (_02) First visual upgrade
Level 10 Maximum Model (_03) Final visual tier, maximum prestige

Exception Categories: Some decorations (Default, Poshu, Huoji, Christmas, Spring Festival, Diamond) maintain the same model across all levels.
