Manor System

System Overview

The Manor System is the building and decoration management component that provides players with a customizable living space. Players can place and upgrade landscape decorations to enhance their manor's appearance and accumulate Prosperity points.

Core Statistics

  • Manor Building Requirement: Must build Manor building first
  • Collection Sets with Bonuses: 2 sets (Dinosaur Park, Sunken Treasure)
  • Themed Decoration Groups: 2 groups (Doomsday, Summer)
  • Total Decorations: 418 Manor buildings and decorations available

System Features

  • Visual Theming: Coordinate decorations into cohesive sets
  • Prosperity Tracking: Accumulate points through decoration upgrades
  • Set Bonuses: Themed collections with matching visual styles
  • Manor Building Requirement: "Please build the Manor first"

Collection Sets

Collection sets are special themed groups that reward players with exclusive Quality 5 decorations when completed. These bonus items have significantly higher Prosperity values and unique visual effects.

How Collection Bonuses Work

  • Collect All Set Items: Obtain every decoration in the collection set
  • Upgrade to Required Level: Each set item must reach a specific upgrade level
  • Unlock Bonus Decoration: A unique Quality 5 decoration becomes available
  • Display on Map: Some bonus decorations (like Alpha Dino at level 3+) can be displayed on the world map
  • Enhanced Prosperity: Quality 5 items provide substantially more Prosperity points than standard decorations

Prosperity Mechanics

Prosperity System

System Message: "Thanks to your thoughtful management, your Landscape Manor has gained [X] points of Prosperity. Once desolate places are now vibrant and eye-pleasing."

(The game displays the actual Prosperity points earned where [X] appears)

How Prosperity Works

  • Players earn Prosperity Points by placing and upgrading landscape decorations
  • Higher level decorations contribute more Prosperity
  • Themed set completion may provide bonus Prosperity (system suggests sets have special value)
  • Prosperity reflects overall manor management success
